Overview

This Unreal Engine project is a fully playable 3D Soulslike prototype with emphasis on reusable, scalable gameplay systems. Designed as a systems engineering showcase, it uses industry-standard patterns to create maintainable, high-performance code.

Key Features

  • Modular Components (Component Design Pattern) — Core gameplay elements are separated into interchangeable modules, enabling quick iteration and feature expansion without breaking existing systems.
  • Delegate-Based Notification Systems — Decouples gameplay systems (UI, AI, combat logic) for cleaner communication, making the codebase easier to scale and maintain.
  • Gameplay Ability System (GAS) Combat — Uses Unreal’s GAS to manage abilities, cooldowns, animations, and combat events in a scalable, network-ready way.

SQL Queries, UI, Database Design

Solo Project

C++, SQL, Interbase

This is the first rendition of my passion project, it is a throwing database application for my track and field training. It is a desktop app that can create, update, delete, and read data from the SQL table in Interbase, and calculates the average of the data from each track meet, and the best throw of those inputted. Has 2 seperate tables for shotput, and discus which are my events, and a UI which lets users perform the CRUD actions and view the tables.

Database App
C++, SQL, Interbase

This was a scaled down proof of concept version of my app that I am designing as a database with a GUI. This app I made runs off of interbase and uses SQL queries to input data from my Track and Field meets. You first input the meet name, date, and ID, it then creates a corrosponding row in bot the Shotput and Discus data graphs where I input my distances, in this picture Shotput only has 1 column since I injured myself and could not throw Shotput at that meet. It then calculates the averages of each meet, and the best throw. Now, it is no Google Sheets or Microsoft Excel (yet) but it was very valuable in helping me envision what I will need for the final version of my app (which you can read about lower on this page).

StackyCapy
C++, Unreal Engine

This is a game I imagined after my social media was filled with capybaras for a whole month. I thought stacking them would be super cute and fun, so that is what I made! So far I have finalized the character which serves as the core gameplay mechanics for the game. You can stack capybaras that you find on top of you, as well as swap to the highest stacked one. The capybaras get smaller the more you have stacked. I did this by creating a customized character class for my capybaras with all of the functionalities it needs, as well as attatchable actor objects, and a game mode. This is still a work in progress but I have had so much fun making it!

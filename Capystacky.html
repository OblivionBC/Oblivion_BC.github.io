CapyStacky

Gameplay, Interactions
Character, World

Solo Project

C++, Git,
Unreal Engine

This is my personal Unreal Engine project. This game has the player as a capybara, stacking other capybaras on top of it so that the capybara can advance to the next level. I have designed and implemented the player class which includes error handling, collision handling, abilities, and movement, as well as the core gameplay mechanics. This is planned to be released in the summer of 2024, but is playable currently. The source code for this game is available on my github which you can check out by clicking the git logo on the right of the taskbar.

Overview

This repository demonstrates production-ready patterns, reference implementations and utilities built around Aerospike Graph Service. It was used for customer proof-of-concepts, internal testing, and reference documentation: everything from dataset generation at scale to secure client-service communication and deployment examples on GCP & AWS.

Role
Intern — design & implementation of generators, load balancers, deployment examples, and demos.
Impact
Enabled terabyte-scale POCs and accelerated performance validation for customers and engineers.

Key Features

  • Scale-free multi-type graph generator (Python) — Generates realistic, multi-type graph datasets at high speed (used to produce terabytes of test data). Plain English: lets teams create large, representative datasets quickly so they can test performance and correctness without waiting days.
    Technical: multithreaded writer, configurable schemas & property distributions, writes up to ~1M records/sec in benchmarked environments.
  • Thread-safe Gremlin driver load balancers — Language implementations (Go, JavaScript, Python) that protect backend endpoints from concentrated load.
    Technical: round-robin request distribution, health checks & automatic revival, dynamic endpoint registration for horizontal scalability.
  • Distributed bulk load examples (GCP & AWS) — Reproducible Apache Spark jobs and deployment recipes demonstrating bulk ingestion to Aerospike Graph at scale.
    Plain English: shows how to run large imports reliably in cloud environments, and how to scale the pipeline for real-world datasets.
  • Transport-level security (TLS) examples — Step-by-step guides and scripts for generating certificates (OpenSSL), attaching secrets, and configuring secure channels between clients, Graph Service, and Aerospike DB.
    Technical: mutual TLS examples, certificate rotation notes, and secure deployment tips.
  • Full-stack food delivery demo (Streamlit) — A DoorDash-like demo app showcasing a working API, user flows, and graph visualizations backed by Aerospike Graph.
    Purpose: visual demo for product teams and customers to explore graph features and common query patterns.

Tech Stack

Primary: Aerospike Graph, Aerospike DB, Python, Go, JavaScript (Node). Supporting: Apache Spark, GCP, AWS, Streamlit. Front-end & demo shells: Angular, Tailwind CSS, Firebase.

Python Go Node.js Apache Spark GCP AWS Angular Streamlit

Results & Notes

  • Generator benchmarks demonstrated sustained multi-threaded writes at high throughput in controlled tests.
  • Load balancer implementations reduced single-endpoint overloads and improved end-to-end query reliability for stress tests.
  • Cloud bulk-load examples were used as reference for two customer POCs and internal performance runs.

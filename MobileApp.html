Mobile Username Generator

UI/UX, Algorithms,
Validation and Authentication

Solo Project

React Native, Javascript, Yup, Formik

This is a mobile username generator built using React Native that uses a variety of algorithms to generate a unique username based on the user's input. The app also has a validation and authentication system using Yup to ensure that the user's input is valid and secure, and the form is styled using Formik.
